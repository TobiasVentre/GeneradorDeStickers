# GeneradorDeStickers

## Core (CLI)
- `npm run build`
- `npm test`

## Frontend Web (nuevo)
El frontend vive en `frontend/` y usa TypeScript + Vite, separado del core.

1. Instalar dependencias del frontend:
   - `cd frontend`
   - `npm install`
2. Ejecutar en local:
   - `npm run dev`
3. Build:
   - `npm run build`

Tambien puedes correr desde la raiz:
- `npm run dev:web`
- `npm run build:web`
